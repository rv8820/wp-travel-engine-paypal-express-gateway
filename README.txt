PayPal Express Checkout Payment Gateway is an extension for WP Travel Engine that allows you to accept trip and tour payments through PayPal Express.

== Description ==

Tour operators and travel agencies can enable the PayPal Express Checkout system for trip booking by setting up PayPal Express Account with key details - Client ID and PayPal Secret.
